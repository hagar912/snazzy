Introduction
------------

This module improves the usability of the term overview list. If you have too many taxonomy terms, this module will reduce your time to edit the term.


Installation
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.
